## üéì Key Concepts to Understand

### 1. Dynamic Job Fetching
**Read**: `DJINNI-ARCHITECTURE.md` ‚Üí Data Flow Diagram

Jobs are fetched from Djinni.co API on-demand, not stored in database.

### 2. Intelligent Caching
**Read**: `DJINNI-ARCHITECTURE.md` ‚Üí Cache Flow

Jobs are cached for 30 minutes to reduce API calls.

### 3. Graceful Degradation
**Read**: `DJINNI-ARCHITECTURE.md` ‚Üí Error Handling Tree

If API fails, shows default positions instead of error.

### 4. Async/Await Pattern
**Read**: `DJINNI-INTEGRATION.md` ‚Üí Backend Implementation

Uses Python async/await for non-blocking HTTP requests.

### 5. React Hooks for Async
**Read**: `DJINNI-INTEGRATION.md` ‚Üí Frontend Implementation

Uses useState and useEffect for data fetching.
